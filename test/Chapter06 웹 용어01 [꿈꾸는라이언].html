웹 및 인터페이스 기술

  • Hypertext (하이퍼텍스트)
  • 다른 문서나 그림으로 이동할 수 있는 연결(링크)을 가지고 있는 텍스트입니다.
  • HTML (Hypertext Markup Language)
  • 하이퍼텍스트 및 웹 페이지를 만들 수 있는 마크업 언어입니다.
  • URL (Uniform Resource Locator)
  • 인터넷 상에서 특정 웹 페이지의 위치를 나타내는 데 사용되는 문자열입니다.
  • MIME (Multipurpose Internet Mail Extensions)
  • 이메일에서 텍스트 외 이미지, 비디오, 오디오 등 여러 유형의 파일을 전송 가능하게 하는 표준입니다.
  • SOAP (Simple Object Access Protocol)
  • 다른 컴퓨터 내 데이터나 서비스를 호출하기 위한 통신 규약 프로토콜로, XML 기반 메시지를 다른 컴퓨터 네트워크 상에서 교환하는 데 사용됩니다.
  • WSDL (Web Services Description Language)
  • 웹 서비스에서 제공하는 서비스에 대한 정보를 XML 기반으로 기술하는 언어입니다 (서비스 제공 장소, 메시지 포맷, 프로토콜 등).
  • UDDI (Universal Description Discovery and Integration)
  • 전역 비즈니스 레지스트리로, 웹 서비스 제공자가 자신들의 서비스를 등록/저장하고 소비자들이 원하는 서비스를 탐색할 수 있도록 하는 온라인 저장소입니다.
  • XML (eXtensible Markup Language)
  • HTML 문법의 비호환성과 SGML의 복잡성을 해결하기 위해 개발된 확장 가능한 마크업 언어입니다.
  • AJAX (Asynchronous Javascript and XML)
  • Javascript와 XML을 이용하여 비동기식으로 웹 페이지의 일부 콘텐츠만 Reload 해오는 방식으로, 다양한 작업을 웹 페이지에서 구현 가능하게 합니다.
  • JSON (JavaScript Object Notation)
  • **속성-값 쌍 (Attribute-Value Pairs)**으로 이루어진 데이터 오브젝트를 전달하기 위해 사용하는 개방형 표준 포맷으로, AJAX에서 많이 사용되며 XML을 대체하는 주요 데이터 포맷입니다.

인터페이스 구현 검증 도구

  • xUnit
  • Java, C++, Net 등 다양한 언어를 지원하는 단위 테스트 프레임워크입니다.
  • JUnit
  • 자바 프로그래밍 언어용으로, 같은 테스트 코드를 여러 번 작성하지 않게 하는 유닛 테스트 프레임워크입니다.
  • Selenium (셀레니움)
  • 다양한 브라우저 및 개발 언어를 지원하는 웹 애플리케이션 테스트 프레임워크입니다.

클라우드 서비스 모델

  • IaaS (Infrastructure as a Service)
  • 물리적 자원(서버, 네트워크, 스토리지)을 가상화하여 제공하는 클라우드 서비스 모델입니다.
  • PaaS (Platform as a Service)
  • 응용 프로그램 개발 시 필요한 플랫폼 (OS, 미들웨어, 런타임 환경)을 제공하는 클라우드 서비스 모델입니다.
  • SaaS (Software as a Service)
  • 사용자에게 제공되는 소프트웨어 (데이터, 앱)를 가상화하여 제공하는 클라우드 서비스 모델입니다.
  • BaaS (Blockchain as a Service)
  • 블록체인 기반 서비스 개발 및 관리의 편의성을 제공하는 클라우드 서비스 모델입니다.
  • DaaS (Desktop as a Service)
  • 각종 업무용 운영체제 및 소프트웨어 등을 클라우드 방식으로 제공하는 서비스입니다 (재택 근무 활용).
  • FaaS (Function as a Service)
  • 사용자가 하부 구조의 복잡한 빌드/유지보수 없이 애플리케이션 기능을 개발, 실행, 관리할 수 있도록 플랫폼을 제공하는 클라우드 서비스 모델입니다.

기타 응용 용어

  • EAI (Enterprise Application Integration)
  • 기업 내 운영되는 플랫폼 및 애플리케이션 간의 정보 전달, 연계, 통합을 수행하는 시스템입니다 (Point to Point, Hub&Spoke, Message Bus, Hybrid 형태로 구성).
  • FEP (Front-End Processor)
  • 입력 데이터를 프로세스가 처리하기 전에 미리 처리하여 프로세스 처리 시간을 줄여주는 프로그램입니다.
  • DRM (Digital Rights Management)
  • 콘텐츠 제공자/분배자/소비자를 관리하며, 암호화, 키 관리, 식별 기술 등을 통해 콘텐츠의 저작권을 통제하고 보호하는 기술 및 관리 시스템입니다.
  • 클리어링 하우스 (Clearing House)
  • DRM 시스템에서 사용 권한, 라이선스 발급, 결제 관리를 수행하는 요소입니다.

UNIX / LINUX 기본 명령어

  • cat
  • LINUX 기본 명령어 중 파일 내용을 화면에 표시하는 명령어입니다.
  • chdir
  • LINUX 기본 명령어 중 현재 사용할 디렉터리 위치를 변경하는 명령어입니다.
  • chmod
  • LINUX 기본 명령어 중 **파일의 보호 모드(권한)**를 설정하는 명령어입니다.
  • cp
  • LINUX 기본 명령어 중 파일을 복사하는 명령어입니다.
  • rm
  • LINUX 기본 명령어 중 파일을 삭제하는 명령어입니다.
  • fork
  • LINUX 기본 명령어 중 새로운 프로세스를 생성하는 명령어입니다.
  • exec
  • LINUX 기본 명령어 중 새로운 프로세스를 수행하는 명령어입니다.
  • getpid
  • LINUX 기본 명령어 중 자신의 프로세스 아이디를 호출하는 명령어입니다.
  • getppid
  • LINUX 기본 명령어 중 부모 프로세스 아이디를 호출하는 명령어입니다.
